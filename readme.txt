Wootb.dialog.plugin

This is a plugin for adding a dialog with slide animation to the screen.


 <div id="boxpopup" class="box">
        <a onclick="$('#boxpopup').WootbDialog('close');" class="boxclose"></a>
        <div id="content">
            dialog content 
	</div>
 </div>

<button type="button" onclick="$('#boxpopup').WootbDialog('show');">dialog</button>
